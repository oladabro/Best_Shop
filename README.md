# Best Shop
A responsive one page website as part of Coders Lab workshop

## Features

- Responsive Web Design, breakpoints at:
  - mobile (<= 767px)
  - tablet (768px-1023px)
  - desktop (>= 1024px)
- Smooth scrolling,
- Animations (hamburger and drop-down menu)

## Technologies

- JS
- SCSS, RWD
- HTML, CSS
